# Required Media Assets

Add the production-ready media files to this directory before deployment.

| File name | Description |
|-----------|-------------|
| marble-envelope.png | Luxurious marble envelope artwork for the wax seal interaction. |
| floral-panel.png | Invitation card floral background revealed after the seal melts. |
| golden-hour-piano.mp3 | Nasheed / instrumental ambience (only plays after interaction). |
| sparkles.mp4 | Looping sparkle overlay video for background ambience. |
| wax-seal-gold.png | Default wax seal artwork displayed on the envelope. |
| waxseal_rosegold.png | Optional rose gold wax seal variation. |
| waxseal_emerald.png | Optional emerald wax seal variation. |
| waxseal_flower.png | Optional floral wax seal variation. |
| emblem-gold.png | Circular emblem background for the top navigation crest. |

These files are intentionally excluded from version control. Coordinate with the design team to supply them before running the final build.
